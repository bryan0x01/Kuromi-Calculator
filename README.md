# Kuromi Calculator
Calculator built with **HTML**, **CSS**, and **JavaScript**.

It can do:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Live Deme
https://bryan0x01.github.io/Kuromi-Calculator/

## Features
- Click the buttons or use your keyboard to type numbers and operators.
- Shows the current number and the result on the screen.
- `C` clears everything.
- `⌫` deletes the last digit.
- `+/-` changes the sign of the number.
- Handles division by zero by showing **"Error"** instead of breaking the app.

## Files
- `index.html` – structure of the calculator
- `styles.css` – styles and grid layout
- `script.js` – JavaScript logic
- `README.md` – this file

## How to use
1. Open `index.html` in your browser.
2. Type numbers using your keyboard or click the buttons.
3. Press `=` or `Enter` to see the result.
4. Use `C` to clear everything and start again.
